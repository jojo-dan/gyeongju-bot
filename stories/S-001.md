# S-001: 웹앱 단순화 — 일정 전용 뷰어로 정리

**Status**: IN_PROGRESS

**Linked tickets**: T-001, T-002, T-003

---

## Story

가족이 "일정만 보는" 단순한 웹앱으로 정리한다.
지도 뷰와 관리(편집) 기능을 제거하고, 코드 구조를 정돈한 뒤,
사용자 위치 기반 거리 표시 기능을 추가한다.

## Acceptance Criteria

- [ ] 지도 뷰(카카오맵 렌더링, 마커, 폴리라인) 완전 제거
- [ ] 관리 기능(PIN 인증, 상태/선택/메모 편집 UI) 완전 제거
- [ ] 카카오 SDK는 Places API(좌표 해석)용으로 유지
- [ ] 제거 후 코드 구조 팀 검토 완료
- [ ] 사용자 위치 기반 거리 표시 기능 구현 (기획 → 승인 → 개발)

## Non-goals

- 봇(텔레그램) 측 변경은 이 스토리 범위 밖
- 웹앱 디자인 전면 리뉴얼은 하지 않음 (기존 디자인 시스템 유지)
